Un bot Discord spécial

Hanaki est un bot Discord codé en JavaScript sur Visual Studio Code avec nodeJS, Discord.js et Mongoose

Qu'est-ce qu'il peut faire ?

Modération

Hanaki peut entre autres, bannir et débannir, expulser, muter et démuter un membre. Il peut aussi supprimer et envoyer des messages dans un ou plusieurs salons.

Divertissement

Hanaki peut divertir les membres du serveur à l'aide de divers commandes fun, telles que 8ball ou encore les dés.

Information

Hanaki peut donner divers informations telles que le nombre de membres, de rôles, de salons, de emojies...etc ainsi que la date de création d'un serveur/compte Discord, l'ID, le nom, l'avatar...etc.

Outils et modules

Visual Studio Code

Visual Studio Code (VSCode) est un éditeur de code extensible développé par Microsoft pour Windows, Linux et macOS

Node JS

Node JS est une plateforme logicielle gratuite en JavaScript, plus précisément, un environnement de bas niveau, orienté vers les applications reseau, permetant l'exécution de JavaScript côté serveur.

Discord JS

Discord JS est un module qui permet d'interagir facilement avec l'API de Discord et ainsi executer diverse commandes écrites par un utilisateur.

MongoDB

MongoDB est un système de gestion de base de données orienté documents. Les données sont repartissable sur un nombre infini de serveur et n'ont pas besoin de schéma prédéfini des données.
